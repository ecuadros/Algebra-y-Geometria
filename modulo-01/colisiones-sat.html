Detección de Colisiones (SAT)

El Teorema del Eje Separador (Separating Axis Theorem) determina si dos polígonos convexos colisionan proyectando sus vértices sobre ejes candidatos. Fundamento de los motores de física en Ciencia de la Computación.

El Teorema del Eje Separador (SAT)

Dos formas convexas no colisionan si y solo si existe un eje separador — una dirección sobre la cual las proyecciones de las dos figuras no se solapan.

proyección de v sobre eje n̂: p = v · n̂ (producto punto) Separación existe si: max(proj A) < min(proj B) ó max(proj B) < min(proj A)

Si no existe ningún eje separador en ninguna dirección candidata, las figuras están colisionando.

Algoritmo SAT

  • 1Para cada arista de ambos polígonos, obtener la normal perpendicular (el eje candidato).
  • 2Proyectar todos los vértices de ambas figuras sobre ese eje usando el producto punto.
  • 3Si los intervalos de proyección no se solapan → hay separación → no hay colisión.
  • 4Si todos los ejes solapan → hay colisión. El MTV (vector de traslación mínima) es el eje con menor solapamiento.

¿Por qué el producto punto?

La proyección escalar de un vector v sobre un eje unitario n̂ se calcula directamente con el producto punto:

p = v · n̂ = vₓ·n̂ₓ + vᵧ·n̂ᵧ

El SAT proyecta todos los vértices sobre cada eje para obtener el intervalo [min, max] de cada figura. Si los intervalos no se tocan, ese eje es separador. La complejidad es O(n·m) donde n, m son el número de aristas de cada polígono.

Aplicación CS

Motor de Física para Videojuegos

SAT es utilizado en motores de juego (Unity, Godot, Box2D) para detección de colisiones entre objetos convexos en tiempo real. Es eficiente porque detiene la búsqueda tan pronto como encuentra un eje separador (early exit). Controla el jugador con WASD o las teclas de flecha y observa la respuesta a colisiones con MTV.

  • Unity usa GJK + EPA para formas arbitrarias, pero SAT internamente para polígonos convexos simples.
  • Box2D emplea SAT con manifold de contacto para resolver penetraciones y fricciones.
  • Godot 4 implementa SAT en su physics server 2D para CollisionShape2D.
  • La respuesta por deslizamiento (sliding) proyecta la velocidad sobre el plano de colisión usando el MTV.
