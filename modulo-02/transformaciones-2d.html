Transformaciones 2D con Matrices

Una matriz 2×2 puede rotar, escalar, reflejar y deformar cualquier figura geométrica.

Multiplicación Matriz-Vector

Toda transformación lineal 2D se expresa como Ax = b, donde A es una matriz 2×2 que codifica la transformación.

⎡ a b ⎤ ⎡ x ⎤ ⎡ ax + by ⎤ ⎣ c d ⎦ ⎣ y ⎦ = ⎣ cx + dy ⎦

Las columnas de A indican hacia dónde van los vectores base î y ĵ.

Matrices de Transformación

R(θ) = ⎡ cos θ −sin θ ⎤ ⎣ sin θ cos θ ⎦
S(sx,sy) = ⎡ sx 0 ⎤ ⎣ 0 sy ⎦
H(hx,hy) = ⎡ 1 hx ⎤ ⎣ hy 1 ⎦

Composición de Transformaciones

Para aplicar múltiples transformaciones, se multiplican las matrices en orden. La última se aplica primero:

M = R(θ) · S(sx,sy) · H(hx,hy)

El orden importa: la multiplicación de matrices no es conmutativa.
