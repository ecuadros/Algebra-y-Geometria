Filtros de Imagen por Convolución

Un kernel de convolución es una pequeña matriz que transforma cada píxel usando sus vecinos.

Operación de Convolución

Para cada píxel en la imagen, se multiplica el kernel por el vecindario 3×3 de ese píxel y se suman los resultados:

output[i][j] = Σm Σn kernel[m][n] · image[i+m, j+n]

El kernel "desliza" sobre cada posición de la imagen. Si la suma del kernel es 1, el brillo se conserva; si es 0, el resultado es una operación de diferenciación.

Kernels Comunes

Identidad (sin cambio):

000 010 000

Nitidez (sharpening):

0-10 -15-1 0-10

Sobel X (bordes verticales):

-101 -202 -101

Normalización y Bordes

Los valores fuera de rango se recortan a [0, 255]. Para kernels de blur, la suma total debe ser 1 (se divide entre el número de elementos activos). Para detección de bordes, la suma es 0 — el resultado muestra cambios de contraste.

Aplicación en Ciencia de la Computación

Detección de Bordes en Visión Computacional

La detección de bordes es el primer paso en muchos algoritmos de visión por computadora: segmentación de objetos, reconocimiento facial, y más. El kernel Sobel calcula el gradiente de intensidad — donde el gradiente es alto, hay un borde. La animación muestra la ventana 3×3 deslizándose por la imagen.
