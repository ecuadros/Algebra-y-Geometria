Cadenas de Markov

Una cadena de Markov modela saltos aleatorios entre estados — la base de PageRank, modelos de lenguaje y más.

Definición

Un sistema con n estados se describe con un vector de estado x_t y una matriz de transición P, donde cada entrada P[i][j] es la probabilidad de pasar del estado j al estado i.

x_t = [p₁, p₂, p₃, p₄]ᵀ Σ pᵢ = 1 (distribución de prob.) P[i][j] = P(j → i)

Cada columna de P debe sumar 1 — es una matriz estocástica.

Evolución del Sistema

Para avanzar un paso en el tiempo, se multiplica la matriz por el vector actual:

x_{t+1} = P · x_t

Después de muchos pasos, la distribución converge al estado estacionario independientemente del estado inicial.

Estado Estacionario

El estado estacionario π es el eigenvector de P con eigenvalor 1:

π = P · π Se encuentra por iteración de potencias: P · P · P · ... · x₀ → π

PageRank usa exactamente este proceso para rankear páginas web.

Los modelos de lenguaje usan cadenas de Markov para predecir la siguiente palabra. Cada estado es una palabra, y la matriz de transición contiene las probabilidades de que una palabra siga a otra. El heat-map muestra las probabilidades de transición entre 5 palabras en español.
