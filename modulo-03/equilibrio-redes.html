Equilibrio de Redes

Las leyes de conservación en redes eléctricas y de flujo se expresan como sistemas de ecuaciones lineales Ax = b.

El sistema Ax = b

En una red de flujo, la conservación en cada nodo produce una ecuación. En conjunto forman un sistema lineal:

A · x = b

A = matriz de incidencia (nodos × aristas)
x = flujos desconocidos en cada arista
b = oferta/demanda externa en cada nodo

Ley de conservación (Kirchhoff de nodos)

  • En cada nodo: flujo entrante = flujo saliente + demanda externa
  • La suma de todas las demandas debe ser cero (conservación global)
  • Si det(A) ≠ 0 → solución única: x = A⁻¹b
  • Si det(A) = 0 → infinitas soluciones o sin solución

Eliminación Gaussiana

Resolvemos Ax = b mediante reducción de filas sobre la matriz aumentada, llevándola a forma escalonada y aplicando sustitución hacia atrás:

[A | b] → operaciones fila → [U | b']
Sustitución hacia atrás → x

Aplicación en CS: Balanceo de Carga en Redes

En redes de computadoras, los enrutadores deben dirigir el tráfico (flujo) desde las fuentes hasta los destinos respetando las capacidades de los enlaces. El mismo sistema lineal modela la asignación de ancho de banda: cada enrutador es un nodo, cada enlace es una arista, y el sistema Ax = b encuentra el flujo en cada enlace. Los enlaces se colorean según su nivel de utilización respecto a la capacidad máxima.
