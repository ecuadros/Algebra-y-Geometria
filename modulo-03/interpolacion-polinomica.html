Interpolación Polinómica

Dado un conjunto de puntos, la matriz de Vandermonde convierte el problema de hallar un polinomio interpolante en un sistema de ecuaciones lineales V·c = y.

Sistema de Vandermonde V·c = y

Para n puntos (x₁,y₁),…,(xₙ,yₙ), exigimos que el polinomio p(xᵢ) = yᵢ. Eso produce el sistema:

⎡ 1 x₁ x₁² ··· x₁ⁿ⁻¹ ⎤ ⎡c₀⎤ ⎡y₁⎤ ⎢ 1 x₂ x₂² ··· x₂ⁿ⁻¹ ⎥ ⎢c₁⎥ = ⎢y₂⎥ ⎢ · · · · ⎥ ⎢·· ⎥ ⎢··⎥ ⎣ 1 xₙ xₙ² ··· xₙⁿ⁻¹ ⎦ ⎣cₙ⎦ ⎣yₙ⎦ p(x) = c₀ + c₁x + c₂x² + ··· + cₙ₋₁xⁿ⁻¹

Propiedades clave

  • Solución única si y solo si todos los xᵢ son distintos
  • n puntos determinan exactamente un polinomio de grado ≤ n−1
  • det(V) = ∏ᵢ<ⱼ (xⱼ − xᵢ) — producto de Vandermonde
  • La matriz es mal condicionada para n grande (puntos muy separados)
  • Se resuelve con eliminación Gaussiana con pivoteo parcial

Evaluación eficiente: Horner

p(x) = c₀ + x·(c₁ + x·(c₂ + ··· + x·cₙ₋₁)) // En lugar de calcular xᵏ n veces, // se anida: O(n) multiplicaciones

Aplicación: Reconstrucción de Trayectorias en Videojuegos

Los motores de videojuegos almacenan solo unos pocos keyframes (posiciones clave) para los movimientos de personajes, cámaras y proyectiles. En tiempo de ejecución, la interpolación polinómica reconstruye la trayectoria suave entre esos fotogramas. Se interpolan las componentes X(t) e Y(t) por separado, cada una con su propio polinomio, y se combinan para obtener la curva 2D. Arrastra los keyframes para ver cómo cambia la trayectoria en tiempo real.
